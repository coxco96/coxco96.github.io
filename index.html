Maps

North American Breweries

This static map is one of my earliest mapping projects. It uses a hexagonal grid to show the number of breweries per 50 square kilometers throughout North America. In this project, I learned how to use design elements on static maps within QGIS.

Kentucky Opioids: The Sackler Family

This "scrollytelling" style web page features two maps presented in a journalistic style. I used R/RStudio to process a very large data file to make these interactive web maps. I made the maps using Leaflet with Javascript, and I created the "scrollytelling" experience using scrollama.js.

Patterns of U.S. Energy Production

This interactive map shows each hydro, wind, natural gas, and solar energy plant in the U.S. I created popup tooltips to show more information about each plant upon hover (or tap on mobile). I used circles proportionate to capacity in order to visually show energy source density. I incorporated design elements that allowed for a visualization of a lot of data without feeling cluttered. This map also includes interactive zoom capability.

Preview PATTERNS OF U.S. ENERGY PRODUCTION

Visit interactive map.

U.S. Rent Prices by County

This interactive map shows average rent price by county in the U.S. I used the following packages in Javascript to create this map: Leaflet, jQuery, and Simple Statistics. With this map I demonstrate using statistical ckmeans breaks in order to cluster data for optimized visualization.

Preview U.S. Rent Prices by County

Visit interactive map.

2001-2015 U.S. Unemployment Rates by County

This map visualizes U.S. unemployment rates from 2001-2015 by giving the user a UI slider bar, providing contrast between years. This especially highlights the 2008 recession and its economical impact.

DENVER, CO: Places of Interest

This is a map of Denver, Colorado featuring all areas with an after school program, a fire station, and a restaurant within 1 mile. The map features a UI element that allows the user to toggle on or off any of the three place categories.

About me

A few things to know

    As a current graduate student in the University of Kentucky's New Maps Plus Program, I'm learning and growing new skills every week, usually faster than I can update this website.

    In May 2022, I began a data fellowship with the McClatchy DC Bureau. Much of the visualization work you see above reflects the work I've done in this role. I've grown immensely my skills immensely through this fellowship, and it is my intention to continue into a career in graphics/data journalism.

    I'm still developing my interests and finding my niches, a process which is both long and invigorating. A few things I know I care deeply about and hope to report on in the future include accessibility barriers to tech, the opioid crisis (see my story on Kentucky above), LGBTQ+ discrimination on Christian college campuses and discrimination against marginalized groups, particularly in healthcare settings.

    As my graduate program is fully online and part-time, I'm open for hire. Please contact me via email or Twitter (see below)!
